## 배열을 만드는 방법

- 배열로 구성하려는 데이터들의 타입
- 배열로 구성하려는 데이터들의 최대 갯수

```java
new 데이터타입[크기]

new int[10] ----- 0 으로 자동 초기화가 일어난다.(타입별 기본값)
new char[26] ----- '\u0000'
new double[5] ----- 0.0
new long[1] -----0L
new boolean[10] ----- false
```

쭉 이어진 메모리방을 할당하게 된다.

Java의 경우 할당된 데이터 크기는 변하지 않는다.
